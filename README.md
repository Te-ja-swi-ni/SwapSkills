# SkillSwap Platform Walkthrough

Welcome to **SkillSwap**, a premium peer-to-peer learning platform designed to connect individuals seeking to exchange skills through mentorship and real-time collaboration.

## 🚀 Key Features

### 1. **Dynamic Skill Discovery**
- **Search & Filter:** Users can search for specific skills (e.g., "React", "Spanish", "Figma") or browse by popular categories.
- **Skill Cards:** Detailed user profiles displaying current "Teaching" and "Learning" goals, ratings, and exchange history.

### 2. **Real-time Messaging**
- **Interactive Chat:** A sleek, glassmorphism-inspired chat interface for seamless communication.
- **Instant Feedback:** Send and receive messages in real-time to coordinate swap sessions.
- **Status Indicators:** See when mentors are online and ready to help.

### 3. **Premium Design System**
- **Dark Mode Aesthetic:** A deep navy background with vibrant purple and blue gradients for a modern look.
- **Glassmorphism:** Elegant UI elements with background blur and subtle borders.
- micro-animations: Powered by `framer-motion` for smooth page transitions and hover effects.

---

## 🛠️ Technical Stack

- **Frontend:** React + Vite
- **Styling:** Vanilla CSS (CSS Variables, Flexbox/Grid)
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Routing:** React Router DOM

---

## 📂 Project Structure

- `src/App.jsx`: Main routing and core layout.
- `src/index.css`: Global design system and theme variables.
- `src/MockData.js`: Centralized mock data for users and conversations.
- `index.html`: SEO-optimized entry point.

---

## 🏁 How to Run

1.  **Install Dependencies:**
    ```bash
    npm install
    ```
2.  **Start Development Server:**
    ```bash
    npm run dev
    ```
3.  **Explore the App:**
    Navigate to `http://localhost:5173/` to see the live platform.
